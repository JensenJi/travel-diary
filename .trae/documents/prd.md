
## 1. 产品概述
根据参考图片样式创建个人简历网站，将所有英文内容翻译为中文，打造现代化、专业的个人展示页面。
- 主要目的是展示个人信息、技术技能、工作经历和联系方式
- 目标用户为招聘方、潜在合作伙伴或需要了解个人背景的访客

## 2. 核心功能

### 2.2 功能模块
1. **首页**: 英雄区、导航栏、关于我、技术技能、工作经历、联系方式&amp;二维码、项目展示、页脚

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 首页 | 导航栏 | 包含"关于"、"联系"、"项目"等导航链接 |
| 首页 | 英雄区 | 个人介绍、头像、技能标签展示 |
| 首页 | 关于我 | 个人简介和开发理念 |
| 首页 | 技术技能 | 前端、样式工具、设计与用户体验等技能分类展示 |
| 首页 | 工作经历 | 展示工作时间、公司名称、职位和工作内容 |
| 首页 | 联系方式&amp;二维码 | 邮箱、电话、位置、社交链接、二维码和消息表单 |
| 首页 | 项目展示 | 展示特色项目，包含项目名称、描述和技术栈 |
| 首页 | 页脚 | 版权信息和网站说明 |

## 3. 核心流程
用户访问首页后，可以浏览个人简介、技能、经历和项目，通过联系方式模块与简历持有者取得联系。
```mermaid
flowchart TD
    A["访问首页"] --&gt; B["浏览个人信息"]
    B --&gt; C["查看技能/经历/项目"]
    C --&gt; D["通过联系方式联系"]
```

## 4. 用户界面设计
### 4.1 设计风格
- 主色调：深色背景（近黑色），搭配蓝紫色渐变作为强调色
- 按钮样式：圆角按钮，带有渐变色和发光效果
- 字体：现代无衬线字体，标题与正文形成层级对比
- 布局风格：卡片式布局，模块化设计，内容分区清晰
- 图标风格：简约线条图标，带有荧光色调，符合现代科技感

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 首页 | 英雄区 | 深色背景、渐变色标题、圆形头像、胶囊式技能标签 |
| 首页 | 关于我/技能/经历 | 卡片式容器、玻璃态效果、渐变色标题 |
| 首页 | 联系方式&amp;二维码 | 标签对齐格式统一、二维码区域、消息表单 |
| 首页 | 项目展示 | 项目卡片、悬停动画效果、技术标签 |

### 4.3 响应性
采用桌面优先设计，同时支持移动端适配，在不同屏幕尺寸下保持良好的可读性和美观性。
